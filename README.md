# 🔍 Fake News Detector

AI powered news verification tool built with LangChain + Mistral API

## 🚀 What it does
- Paste any news article
- AI analyzes it instantly
- Get REAL/FAKE/MISLEADING verdict
- Confidence score + detailed analysis

## 🛠️ Tech Stack
- Python
- LangChain
- Mistral API
- Streamlit

## ▶️ How to Run
pip install -r requirements.txt
streamlit run app.py
